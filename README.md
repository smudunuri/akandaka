akandaka
========
A cross platform secure password manager

A javafx based secure store for passwords, notes, yada yada...

<h6>Runtime Requirements</h6>
JRE 8 <br>
JavaFX 8 runtime <br>
JCE Unlimited strength jurisdiction policy <br>


<h6>How to Run</h6>
java -jar akandaka.jar<br>
